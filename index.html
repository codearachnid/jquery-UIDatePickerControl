jQuery.UIDatePickerControl

A simple iOS-esc jQuery plugin to create a UIPickerControl. Currently implemented as a calendar picker with intentions to abstract.

jQuery.UIDatePickerControl

This is a live demo, test it by clicking and dragging (or load on mobile and swipe).

Requires

  • jQuery (tested 1.8.3)
  • jQuery.nearest v1.2.0
  • jQuery.Event.Swipe v0.5
  • jQuery.Event.Move v1.3.6
  • jQuery.dateFormat v1.0
